# @vuepress-reco/vuepress-plugin-pagation

Pagation plugin for vuepress-theme-reco or other vuepress theme

## 选项

### total

数据总数量。

- 类型：`number`
- 默认值：`10`

### perPage

每页多少条。

- 类型：`number`
- 默认值：`10`

### currentPage

当前是第几页。

- 类型：`number`
- 默认值：`1`

## 事件

### getCurrentPage

获取当前是第几页。

- 参数 currentPage: 当前页码
